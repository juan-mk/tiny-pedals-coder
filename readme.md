Tiny Pedals Big Sound
Description

Tiny Pedals Big Sound is an educational static website focused on guitar effects pedals. The project showcases different types of pedals, their features, designs, and multimedia content through a responsive and user-friendly interface.

This website was developed as a web development project, inspired by the world of boutique guitar effects pedals and their creative designs.

Features
Responsive institutional website with 5 sections:
Home
About
Gallery
Media
Contact
Mobile-first responsive design
Custom navigation and footer
Product showcase with guitar effects pedals
Image gallery and multimedia section
Contact form layout
Social media integration
Floating WhatsApp button
Custom favicon and branding
Technologies Used
HTML5
SASS / SCSS
Bootstrap 5
Font Awesome
Google Fonts



Design
The website follows a dark and minimalist aesthetic inspired by guitar pedal brands, using a black and white color palette with a focus on clean layouts and visual presentation.

Deployment
The project was deployed using Vercel and connected to GitHub for version control.
